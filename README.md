# 物流タイピング

物流会社で働く社会人2年目程度の実務理解を想定した、日本語・英語の長文タイピング練習アプリです。HTML / CSS / JavaScriptのみで動作し、外部ライブラリやサーバーは不要です。GitHub Pagesにそのまま配置して公開できます。

## 機能

- ブラウザだけで動く静的Webアプリです。
- 練習文章はすべて300文字以上の長文です。
- 入力した文字に応じて、お手本文章側をリアルタイムに色分けします。
  - 正しい文字：緑
  - 間違った文字：赤
  - 未入力文字：グレー
- タイピング開始から完了までの時間を計測します。
- 文字数/分、正答率、ミス数、入力文字数を表示します。
- 速度に応じて S / A / B / C / D / E のランクを表示します。
- 「文章を変更」ボタンで同じカテゴリの次の問題に進めます。
- 「リセット」ボタンで現在の問題を最初から練習できます。

## カテゴリ

各カテゴリに5問ずつ、合計15問の練習文章を収録しています。

1. 物流全般
   - B/L確認、ETD/ETA変更、Free Time、Demurrage、Detention、危険品輸送、港混雑、運賃変動などを含む日本語の実務文章です。
2. 英語メール
   - 船会社、フォワーダー、通関業者、海外代理店との調整を想定した英文メールです。
3. 物流全般英語版
   - 物流実務の判断・確認・調整を英語で説明する長文です。

## 使い方

1. `index.html` をブラウザで開きます。
2. カテゴリを選択します。
3. 表示されたお手本文章を、下の入力欄にそのまま入力します。
4. 入力を開始するとタイマーが自動で動きます。
5. 入力が完了すると、時間・速度・正答率・ランクが表示されます。
6. 別の文章に進む場合は「文章を変更」を押します。

## 練習文章を追加・編集する方法

練習文章データは `script.js` の一番上にある `practiceTexts` にまとめています。

```js
const practiceTexts = {
  "物流全般": [
    "ここに300文字以上の文章を入れます。",
    "2問目の文章を入れます。"
  ],
  "英語メール": [
    "Subject: ..."
  ],
  "物流全般英語版": [
    "In daily forwarding operations, ..."
  ]
};
```

### 編集ルール

- 1カテゴリにつき最低5問を維持してください。
- 1問あたり最低300文字以上にしてください。
- 短文を使いたい場合は、複数文をつなげて1回の練習合計が300文字以上になるようにしてください。
- カテゴリ名を変更した場合は、`index.html` の `<select id="categorySelect">` の `<option>` も同じ名前に変更してください。
- 新しい物流ニュースを反映する場合は、`practiceTexts` 内の該当カテゴリの文章を差し替えるだけで画面に反映されます。
- 文章には、単なる用語説明だけでなく、実務で必要な判断・確認・調整を含めると練習効果が高くなります。

## GitHub Pagesで公開する方法

1. このリポジトリをGitHubにpushします。
2. GitHubのリポジトリ画面で `Settings` を開きます。
3. 左側メニューの `Pages` を選択します。
4. `Build and deployment` の `Source` を `Deploy from a branch` にします。
5. 公開したいブランチを選び、フォルダは `/ (root)` を選択します。
6. `Save` を押すと、数分後にGitHub PagesのURLが表示されます。

## ローカル確認

リポジトリ直下で以下を実行すると、ローカルサーバーで確認できます。

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

ブラウザで `http://127.0.0.1:4173/` を開いてください。
